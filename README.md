## Easyplant - A web application for plant Shoping

Easyplant is a web application for plant Shoping. Users can buy plants. Admin can add, edit and delete plants.
Users can add plants to cart and buy them.

## Installation

```bash
git clone
cd easyplant
npm install
npm start
```
